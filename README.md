# Daily Checklist Dashboard

A productivity dashboard built with Streamlit to track daily goals, OKRs, and personal progress.

## Features

- Daily task tracking
- OKR monitoring
- Progress visualization
- Data validation
- Clean UI dashboard

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib

## Project Structure

app/ → main application code
data/ → datasets
tests/ → test files
docs/ → documentation

## Installation

Clone the repository

```bash
git clone https://github.com/KAMERAVAMSHI/daily-checklist-dashboard.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the app

```bash
streamlit run app/dashboard.py
```
